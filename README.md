# Spinner

A decision wheel for when you can't decide. Add your options, hit SPIN, and let physics-flavored randomness pick for you.

- **Editable options** — add, remove, or edit any option in place. Your list is saved in the app (localStorage), so it's still there next time.
- **Satisfying spin** — 4–7 full turns with a smooth ease-out, distinct colors per segment, pointer at the top, and a result banner when the wheel settles.
- **Defaults to the important questions**: Build, Ship, Demo, Touch grass.

A [Shoebox](https://github.com/shoebox-apps) bundle: a single `index.html` with inline CSS/JS, no build step, no external libraries, no permissions. It runs in the Shoebox host's locked-down WebView — or in any browser.

## Run it

- On your phone: add `https://shoebox-apps.github.io/app-spinner/` in the Shoebox host.
- Locally: `python3 -m http.server` in this directory, then open `http://localhost:8000`.

## Make it yours

The whole app is one file. Change the default options, the palette, the max spin turns — anything.
